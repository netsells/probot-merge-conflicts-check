# Netsells PR Merge Conflict Checker Bot

> A GitHub App built with [Probot](https://github.com/probot/probot) that notifies when a Pull Request has merge conflicts with the base branch.

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```
